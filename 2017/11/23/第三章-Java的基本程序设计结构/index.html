  1. Java应用程序分析

    分析这道程序：

    1. 首先Java区分大小写，拼写错误，会导致程序无法运行。
    2. 关键字public成为访问修饰符，用于控制程序的其他部分对这段代码的访问级别。关键字class表明Java程序中的全部内容都包含在类中。此处只需要将类作为一个加载程序逻辑的容器，程序逻辑定义了应用程序的行为。类是构建所有Java应用程序和applet的构建块。

  2. 注释

    第三种注释可以自动生成文档。以/**开始，以x/结束。

  3. 数据类型

    Java是一种强类型语言。每个变量都必须声明一种类型。Java中有8中基本类型(primitive type)，4中整型，2种浮点型，1种用于表示Unicode编码的字符单元的字符类型char和1种用于表示真值的boolean类型。

    PS:Java中的“大数值”，big number，可以表示任意精度的算书包，但其是一个Java对象，而不是Java类型。

    1. 整型

      整型用于表示没有小数部分的数值，可以是负数。Java提供了4种整型。

      | 类型 | 存储空间 | 取值范围 |
      | :—: | :–: | :———-: |
      | int | 4字节 | -20亿~20亿 |
      | short | 2字节 | -32768~32767 |
      | long | 8字节 | 很大很大 |
      | byte | 1字节 | -128~127 |

      int最常用，long用于超大数值，byte和short用于特定场合，如文件流或数组计算时。

      Java中，整型的范围与Java运行平台无关，其在所有机器都能得到相同的运行结果，解决了移植问题。

      长整型数值有一个后缀L，十六进制前缀0x，八进制前缀0，但八进制容易混淆，最好不用八进制常数。

      自Java7开始，加前缀0b表示二进制数，还可以在数字字面量加下划线，目的是为了易读，和注释效果一样。

      PS:c/c++中，int,long等的数值与目标平台有关。Java没有任何无符合(unsigned)形式的类型。

    2. 浮点类型

      浮点类型用于表示有小数部分的数值。

      | 类型 | 存储需求 | 取值范围 |
      | :—-: | :–: | :————–: |
      | float | 4字节 | 和int差不多，有效位数6~7位 |
      | double | 8字节 | 同样很大很大，有效位数15位 |

      double是双精度数值，因为其类型的数值精度是float类型的两倍。大部分情况采用double类型，float只适合精度低的场合，如需要单精度数据的库，存储大量数据时。

      float类型的数值后缀为F,f，没有后缀默认为double，也可以加后缀d,D。

      所有浮点数计算都遵循IEEE 754规范。正无穷大，负无穷大，NaN(非数值)表示溢出和出错情况。常量Double.POSITIVE_INFINITY，Double_NEGATIVE_INFINITY和Double.NaN表示这三种情况。用Double.isNaN来判断其值是否出错。

      如果在数值计算中不允许有任何舍入误差，可以用BigDecimal类j精确表示。

    3. char类型

      char类型用来表示单个字符，通常用来表示字符常量。其字面量值需用单引号‘’括起来。如‘A’表示编码值为65所对应的字符常量。而“A”表示的是包含字符A的字符串。char类型对应Unicode编码，可以表示为十六进制值，范围为\u0000到\Uffff。

      转义序列\u(其后必须跟4个16进制数)表示Unicode代码单元的编码，所有的转义序列都可以出现在加引号的字符字面量或字符串中。而只有\u才可以出现在字符常量或字符串的引号之外。

      特殊字符的转义序列符：(Unicode转义序列会在解析代码前得到处理)

    4. Unicode和char类型

      之前各国都有自己的传统字符编码机制，美国ASCII、西欧ISO 8859-1、中国的GB18030等，由此产生问题一、同样的代码值，不同的编码对应不同的字母；二、采用打字符集的语言，其编码长度可能不同，如有的字符采用单字节，有的则两个或更多。

      设计Unicode编码就是为了统一和规范标准，以此解决问题。Java采用16位的Unicode字符集，Unicode字符超过了65536个，因为增加了大量的汉语、日语和韩语中的表意文字。现在16位的char类型已不能满足所有的Unicode字符的需要了。

      在Java中，char类型描述了UTF-16编码中的一个代码单元。强烈建议不要再程序中使用char类型，最好将字符串作为抽象数据类型处理。

    5. boolean类型

      boolean类型有false和true两个值，用来判定逻辑条件。整型值和布尔值不能相互转换。

      PS:C++中，0相当于false，非0相当于true，这在Java中并不成立。

  4. 变量

    在Java中，每个变量都必须声明自己的类型 ge

    类型 变量名; 声明是一条完整的语句，必须以分号结束。

    变量名命名规则：

    • 必须以字母开头，由字母或数字构成
    • 字母包括A~Z，a~z、_、$，数字包括0~9，或者表示字母和数字的任何Unicode字符。
    • 变量名大小写敏感，长度没有限制，所以字符都有意义

    可在一行中声明多个变量，不过不推荐，逐一声明每个变量可以提高程序的可读性。

    1. 变量初始化

      变量声明后，需要用“=”赋值后才能使用。

      • 变量声明可以放在代码任何地方
      • Java中变量声明尽可能靠近变量第一次使用的地方，便于阅读
      • C/C++中区分变量的声明与定义，没赋值前是声明，赋值后是定义。Java不区分声明与定义

    2. 常量

      在Java中利用关键字final指示常量，表示这个变量只能被赋值一次。习惯上，常量名全大写。

      如果希望某个常量可在类中多个方法使用，这样的常量称为类常量，用static final修饰，如果声明为public，那么其他类也可以使用这个常量

  5. 运算符

    Java中参与/运算的两个操作数都是整数，表示整数除法，否则就表示浮点除法。整数的取余操作用%表示。整数被0除将会产生异常，而浮点数被0除将会得到无穷大或NaN结果。

    简化的二元赋值运算符+=、-=、*=、%=。通常将运算符放在赋值号左侧。

    1. 数学函数与常量

      Math类中，包含了许多常用的数学函数

      如计算数值平方根，可以使用sqrt方法Math.sqrt(x)

      如幂运算，借用pow方法Math.pow(x,a)

      以及常用三角函数sin,cos,tan等

      还有指数函数以及反函数Math.log,Math.log10

      Java还提供了两个用于表示π和e的近似值：Math.PI,Math.E

      Math类中，为达到最快性能，所有方法都使用计算机浮点单元中的例程。如果得到一个完全可预测的结果比运行速度更重要的话，可以使用StrictMath，其使用自由发布的Math库(fdkubm)实现算法，以确所有平台得到相同的结果。

    2. 数值类型之间的转换

      程序运行时，经常会有数值类型间的转换，程序默认自动转化如下

      byte–>short–>int–>long int–>double

      char–>int float–>double都可在无精度丢失的情况下转化。

      int–>float long–>float long–>double都会有精度损失。

      • 如两个操作数有一个是double类型，那另一个操作数就会转换为double类型

      • 否则，如果其中一个操作数时float类型，那另一个操作数就会转换为float类型

      • 否则。。long….long

      • 否则，都将被转换为int类型

        也就是说byte，short，char参与运算时都会被自动提升为int类型运算。

    3. 强制类型转换

      强制类型转换也会造成数据精度丢失，double转化int时，其会通过截断小数部分将浮点值转换为整数，如果想对浮点数进行四舍五入运输，可以用Math.round方法。

      如果试图将一个数值转换但超出目标类型的表示范围，结果就会截断成一个完全不同的值。

    4. 自增自减运算符

      前缀和后缀两种形式，在表达式中，前缀形式会先完成+1，而后缀形式会使用变量原来的值。建议表达式中不要用++。

    5. 关系和boolean运算符

      ==检测相等性 ！= 检测不相等 其他常用的<,>,<=,>=运算符

      &&表示与逻辑运算符，||表示或逻辑运算符，二者都是按照短路方式求值：如果左侧操作数已经能够确定表达式的值，右侧就不必计算了。这种方式推荐使用，避免复杂运算的错误。

      ！是逻辑非运算符

      Java也支持三元运算符？：，如果条件为true就第一个表达式，否则为第二个表达式

      condition?expression1:expression2

    6. 位运算符

      处理整型类型时，可以直接对组成整型数值的各个位完成操作，这意味着可以使用掩码技术得到整数中的各个位。

      位运算符包括：&与，|或，^异或，~非

      这些运算符按位模式处理。

      &和| 运算符应用于布尔值，结果也是布尔值，不是短路运算，左右两侧都要运算

      》和《运算符将二进制进行右移或左移操作。》>运算符会用0填充高位，而》会用符号位填充高位不存在<<<运算符

    7. 括号与运算符级别

      如果不使用()就按照运算符优先次序进行计算，同一级别运算符从左到右的次序进行计算

    8. 枚举类型

      有时候变量的取值只在一个有限的集合内。可以自定义枚举类型，包括有限个命名的值。

      如：enum Size{SMALL,MEDIUM,LARGE}

      也可以声明这种类型的变量：Size s = Size.MEDIUM;

      Size类型的变量只能存储这个类型声明中给定的某个枚举值，或为null值，null表示这个变量没有设置任何值。

  6. 字符串

    概念上讲，Java字符串就是Unicode字符序列。Java没有内置的字符串类型，而是在标准Java类库中提供了一个预定义类String，每个用双引号括起来的字符串都是String类的一个实例

    1. 子串

      String类的substring(x,y)方法可以从字符串提取子串，要提取的字符位置包括x,但不包括y，即范围为[x,y)，优点是容易计算子串长度y-x。

    2. 拼接

      Java允许使用+拼接两个字符串

      当将一个字符串与一个非字符串的值进行拼接时，后者被转换成字符串(任何一个Java对象都可以转换成字符串)

    3. 不可变字符串

      String类没有提供用于修改字符串的方法，Java文档中将String类对象称为不可变字符串。只能通过剪切拼接重新赋值，任何对String的改变都会引发新的String对象产生。String类用字符数组保存字符串，并用final修饰，也保证了String对象的不可变。

      不可变字符串的优点是：编译器可以让字符串共享。这样将各种字符串存放在公共的存储池中，字符串变量指向存储值中相应的位置。如果复制一个字符串变量，原始字符串与复制字符串共享相同字符。Java设计者认为共享效率大于提取和拼接字符串所带来的低效率。

    4. 检测字符串是否相等

      可用equals方法检测两个字符串是否相等(equals注重的是内容相同相同，不需要对比地址是否相同)，如果不区分大小写，可以使用equalsIgnoreCase方法

      不要使用== 运算符检测是否相等！因为==只能确定两个字符串是否放在同一位置，而不能确定字面量是否一致。(当然地址相同，必然相等)。如果虚拟机始终将相同的字符串共享，就可以使用 == 运算符检测是否相等，但实际上只有字符串常量是共享的，而+或substring等操作产生的结果并不是共享的。因此不要用 ==

    5. 空串与null串

      空串“”是长度为0的字符串，可用if(str.length() == 0)或if(str.equals(“”))判断。

      空串是一个Java对象，有自己的串长度0和内容 空。String变量还可以存放一个特殊值，null，表示目前没有任何对象与该变量关联。用if(str == null)判断字符串是否为null。

    6. 代码点与代码单元

      str.length str.charAt()

    7. String API

      • char charAt(int index) 返回给定位置的代码单元。不建议使用此方法
      • int compareTo(String other) 按字典顺序排序，若字符串位于other之前，返回负数，之后返回正数，相等，返回0
      • boolean equals/equalsIgnoreCase() 字符串是否相等
      • boolean startsWith/endsWith(String str) 字符串以str开头或结尾 返回true
      • int indexof(String str/String str,int fromIndex/int cp/int cp ,int fromIndex)返回与字符串str或代码点cp匹配的第一个子串的开始位置，这个位置从0或fromIndex开始计算。如果str不存在，返回-1。
      • int lastIndexof(同上)返回与字符串str……最后一个子串的开始位置，这个位置从串尾或fromIndex开始计算。
      • int length()返回字符串长度
      • String replace(oldstring,newstring)，用newstring 代替oldstring
      • boolean startsWith(str)判断是否以str开始
      • String substring(int int)
      • String toLowerCase()/toUpperCase()返回新字符串，将原始字符串中的大写改小写或小写改大写
      • String trim() 返回新字符串，删除字符串头尾的空格
      • String join() 返回新字符串，用给定的定界符连接所有元素

    8. 构建字符串

      String类型的字符串每次连接新的字符串都会构建一个新的String对象，即耗时又浪费空间，可用StringBuilder类避免这个问题

      用builder.append(str)添加字符，用builder.tostring()构建字符串。

  7. 控制流程

    1. 块作用域

      块是由一对大括号括起来的若干条简单的Java语句，其确定了变量的作用域。一个块可以嵌套在另一个块中，但是，不能再嵌套的两个块中声明同名的变量。

    2. 条件语句

      if(condition)statement

    3. 循环语句

      while(condition)statement

    4. 确定循环

      for是while循环的简化形式

    5. 多重选择：switch语句

      switch语句将从与选项值相匹配的case标签处开始执行，直到遇到break语句或执行到switch语句的结束。如果没有相匹配的case，而有default语句，就执行默认语句。

      case标签支持类型：char,byte,short,int，枚举常量，字符串字面量(j7开始支持的)

    6. 中断控制流程语句

      break在循环中时，可用于退出循环

      continue中断此次循环，继续下次循环

  8. 大数值

    如果基本的整数和浮点数精度不能满足需求，可以用Java.math包提供的BigInteger和BitDecimal，这两个类可以处理包含任意长度数字序列的数值。都可以实现任意精度的整数或浮点数运算。

    可以用valueOf方法将普通数值转换为大数值。不能用算术运算符来处理大数值，只能用add，multiply等处理。

    add()加，subtract减，multiply乘，divide除

    int compareTo(BitInteger/BigDecimal other)如果大数值与另一个大数值相等，返回0，小于返回负数，大于返回正数

    value of(x)返回值等于x的大数值。

    BigInteger mod(other) 返回这个大整数和另一个大整数的和、差、积、商和余数

    BigDecimal divide(BigDecimal other,RoundingMode mode) 5.0 返回这个大实数和另一个大实数的和、差、积、商和余数。要计算商，必须给出四舍五入rounding mode。RoundingMode.HALF_UP就是四舍五入方式。

  9. 数组

    数组是一种数据结构，用来存储同一类型值的集合，通过其下标可以访问数值的每一个值。

    数据类型 [] x;

    可使用new运算创建数组。数组长度可以使常量，也可以是n。数组下标从0开始。

    数字数组，所有元素初始化为0；boolean元素数组为false;对象元素则为null，表示还没存入任何对象。

    如果想获取数组元素的个数可以用array.length

    数组一旦创建，就不能改变大小(尽管可以改变每一个数组元素)。如果需要在运行过程中扩展数组大小，需要数组列表(array list)

    1. for each循环

      即增强for循环

      for(variable:collection)statement。

      定义一个变量variable用于暂存集合中的每一个元素，并执行相应的语句。collection这一集合表达式必须是一个数组或者是一个实现了Iterable接口的类对象。

    2. 数组初始化以及匿名数组

      int [] arr ={x,y}这种表示方法，可以创建数组对象的同时进行初始化赋值

      new int[]{x,y}匿名数组

      Java中，允许数组长度为0。PS：数组长度为0与null不同

    3. 数组拷贝

      Java中允许一个数组变量拷贝给另一个数组变量，这时两个变量将引用同一个数组。

      使用Arrays类的copyOf也可以达到复制效果

      int copyarr[] =Arrays.copyOf(arr,arr.length)

      如果长度大于arr.length，多余元素将按照类型初始化值，如int类型，赋值0，布尔型，为false.如果长度小于arr.lenth，则只拷贝最前面的数据元素。

    4. 数组排序

      Arrays类的sort方法可以对数值型数组进行优化的快速排序，效率较高

      API:

      • static String toString(type[] a) 5.0

        返回包含a中数据元素的字符串 a的类型可以为int long short char byte boolean float或double数组

      • static type copyOf/copyOfRange(type[ ] a,int start,int end) 6

        返回与a相同的一个数组，其长度为length或end-start，数组元素为a的值。PSlength根据实际情况变化

      • static int binarySearch(type[ ] a,int start ,int end , type v)

        采用二分查找搜索算法查找值V，找到成功，返回相应的下标值，否则返回负数r。

      • static void fill(type[] a ,type v)

        将数组的所有元素值设置为v

      • static boolean equals (type[ ] a,type[ ] b)

        如果两个数组大小相同，并且下标相同的元素都对应相等，返回true

    5. 多维数组

      多维数组将使用多个下标访问数组元素，适用于表示表格或更加复杂的排列形式。

      声明一个二维数组：数据类型 [][] [ ]0[ ]数组名;

      也可以简写形式初始化{花括号，花括号，花括号} PS：不这么写无法上传，理解就好。。。

      遍历二维数组时，for each循环不能自动处理二维数组的每一个元素，它是按照一维数组处理的，要想访问二维数组的所有元素，需要使用两个嵌套循环。

      如需要快速打印一个二维数组元素列表，可调用Array.deepToString()

    6. 不规则数组

      Java本质上只有一维数组，多维数组又被称为：数组的数组。

      如b[a]/[c]实际上表示二维数组b有a个子数组，每个子数组的长度为c。

      b[i]表示引用第i个子数组，也就是二维表的第i行，其本身也是一个数组，b[i]0[j]表示引用第i行数组的第j个元素。

      因为可以单独存取数组的某一行，所以可以让这两方交换。

      由此也可以构造数组每行不同长度的不规则数组。
