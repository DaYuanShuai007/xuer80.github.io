屠龙技数据结构与算法算法

第一章-基础数据结构

2017-12-07
数据结构

数据机构即，便于算法操作的组织数据的方法

算法描述一种有限、确定、有效的并适合用计算机程序来实现的解决问题的方法。

一、基础编程模型

我们把描述和实现算法所用到的语言特性、软件库和操作系统特性总称为基础编程模型。我们选择Java来编写程序因为：

  • 程序是对算法精确、优雅和完全的描述
  • 可以通过运行程序来学习算法的各种性质
  • 可以在应用程序中直接使用这些算法

  1. Java程序的基本结构

    Java及现代语言的语法，共同点有以下几点：

    • 原始数据类型：在计算机程序中精确的定义整数、浮点数和布尔值等。其定义包括取值范围和相应的操作，可以组合类似于数学公式的表达式
    • 语句：通过创建变量进行赋值、控制运行流程或引发副作用进行计算。如声明、赋值、条件、循环、调用和返回。
    • 数组：数组时多个同种数据类型的值的集合
    • 静态方法：静态方法可以封装并重用代码
    • 字符串：即一连串的字符，Java内置一些操作
    • 标准输入/输出：是为了程序与外界联系的桥梁
    • 数据抽象：数据抽象封装和重用代码，是我们可以定义非原始数据类型，进而支持面向对象编程

  2. 原始数据类型与表达式

    数据类型就是一组数据和对其所能进行的操作的集合

    如整型、浮点型、布尔型和字符型

  3. 语句

    Java程序由语句组成，语句能够通过创建和操作变量、对变量赋值并控制这些操作的执行流程来描述运算，语句包括

    • 声明语句：创建某种类型的变量，并用标识符为其命名
    • 赋值语句：将某种类型的数值赋值给一个变量
    • 条件语句
    • 循环语句
    • 调用和返回语句

    程序就是由一系列声明、赋值、条件、循环、调用和返回语句组成的。

  4. 数组

    数组能顺序存储相同类型的多个数据。访问数组中的某个元素的方法是将其编号然后索引，编号从0开始。

    创建数组需要三步：

    • 声明数组的名字和类型
    • 创建数组(需指定数组的长度或者元素个数)
    • 初始化数组元素

    数组一经创建，大小就是固定的

    二维数组a[i]0[j]就是第i行第j列的元素。double [][] []0[] a = new double [M]0[N]这样的数组又称为MxN数组。约定第一维是行数，第二维是列数。

  5. 静态方法

    本书中所有Java程序要么是数据类型的定义，要么是一个静态方法库。静态方法又称为函数，是一组在被调用时会被顺序执行的语句。修饰符static将静态方法和实例方法区分开来。

    方法的性质：

    • 方法的参数按值传递：方法处理的是参数的值，而非参数本身。值传递也意味着数组参数将会是原数组的别名(而非拷贝，别名数组与原数组指向同一组数值)
    • 方法名可以被重载：
    • 方法只能返回一个值，但可以包含多个返回语句
    • 方法可以产生副作用：方法返回值可以是void，即没有返回值，但可有输入输出，修改数组等操作。

    递归：方法可以调用自己。递归代码更加简洁、易懂。递归的需注意：

    • 递归总有一个最简单的情况——方法的第一条语句总是一个包含return的条件语句(即第一次循环的判断)
    • 递归调用总是去尝试解决一个规模更小的子问题，这样递归才能收敛到更简单的情况。
    • 递归调用的父问题和尝试解决的子问题之间不应有交集

    使用递归的另一个原因是我们可以使用数学模型来估计程序的性能。

    模块化编程：通过静态方法库来实现模块化编程，可以构造多个静态方法库(模块),一个库中的静态方法也可调用另一个库中定义的静态方法，这样可以带来诸多好处：

    • 程序整体的代码量很大时，每次处理的模块大小仍然适中
    • 可以共享和重用代码而无须重新实现
    • 很容易用改进的实现替换老的实现
    • 可以为解决编程问题建立合适的抽象模型
    • 缩小调试范围

  7. 字符串

    字符串是由一串字符(char类型的值)组成的。String类型的字面量包括一对双引号和其中的字符。

    parseInt/Double 将字符串转换为整数/浮点数

    toString(int/double x) 将整数/浮点数转换为字符串

    Java中在连接字符串时会自动将任意数据类型的值转换为字符串，也可以通过“”将任意数据类型转换为字符串值
